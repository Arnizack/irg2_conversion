# Ordnerstruktur

Die Ordnerstruktur sollte folgendermaßen aussehen:

```
mainfolder
├───Monkeys.csv
|
├───<Patcher Name 1>_Cells.csv
|
├───<Patcher Name 2>_Cells.csv
|
├─── ....
|
├───<Monkey Name 1>_<Datum>
│   ├───<Patcher Name 1>
│   |   ├───<Cells 1>
|   |   |   ├───<>.cfs
|   |   |   └─── ...
|   |   ├───<Cells 2>
|   |   |   ├───<>.cfs
|   |   |   └─── ...
|   |   ├───<Cells 3>
|   |   |   ├───<>.cfs
|   |   |   └─── ...
|   |   └─── ...
|   └───<Patcher Name 2>
│       ├───<Cells 1>
|       ├───<Cells 2>
|       ├───<Cells 3>
|       └─── ...
├───<Monkey Name 2>_<Datum>
│   ├───<Patcher Name 1>
│   |   ├───<Cells 1>
|   |   └─── ...
|   ├───<Patcher Name 2>
│   |   ├───<Cells 1>
|   |   └─── ...
|   └───<Patcher Name 3>
│       ├───<Cells 1>
|       └─── ...
└─── ....

```
